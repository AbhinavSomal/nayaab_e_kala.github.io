How to add enrollment free

  1. Create a Google Form with fields like name, phone, email, selected course, city, and payment status.
  2. Copy the form link and replace every sample Enroll button link in this file.
  3. Check Google Sheets responses so you can manage inquiries and admissions easily.

How to publish free

  1. Create a GitHub account or Cloudflare account.
  2. Upload this HTML file as your website entry file.
  3. Publish it on a free subdomain like yourname.github.io or yourproject.pages.dev.
  4. Later, connect a paid custom domain if you want a more professional brand.
